# AI Hand Gesture Recognition

A real-time AI-powered hand gesture recognition system built using MediaPipe, OpenCV, Streamlit, and WebRTC.

## Features
- Real-time hand tracking
- 21 hand landmark detection
- Thumb-index distance measurement
- Finger angle calculation
- Browser-based live webcam processing
- Streamlit web interface

## Technologies Used
- Python
- MediaPipe
- OpenCV
- Streamlit
- Streamlit-WebRTC
